# Iceland — Clyde & Lucy · July 2026

A pocket field guide for our Ring Road trip (8–16 July 2026), built for phones.

- **Now** — time-aware suggestions (Iceland time): breakfast near last night's guesthouse in the morning, lunch stops at noon, dinner near tonight's bed in the evening.
- **Days** — the full day-by-day plan with check-in details, numbered route stops, and food ideas. Every place links to Google Maps / navigation / its website.
- **Map** — the whole Ring Road with real driving geometry, one color per day.
- **Info** — flights, rental car, all seven beds, and the extras list.
- **Guide** — a chatbot that knows the whole itinerary (OpenRouter, `tencent/hy3:free`). Paste an API key once; it's stored only in the browser.
- **Pronunciation** — tap the 🔊 next to any Icelandic name to hear it spoken and see an English phonetic respelling (plus what the name means). Audio was pre-generated with [Piper TTS](https://github.com/OHF-Voice/piper1-gpl)'s `is_IS-salka` neural voice (trained on Iceland's open Talrómur corpus) and ships as static MP3s — no API, no keys, works offline.

Booking codes and door PINs are lightly obfuscated (base64) and revealed on tap.
